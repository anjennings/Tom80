# Tom80 V2.0![alt text](https://github.com/anjennings/Tom80/blob/Version-2.0/Photos/V2%20Partial.jpg?raw=true)New Features in V2:* CTC for periodic hardware interrupts* PIO for interrupt based bi-directional communication* Parallax Propeller for VGA video, PS/2 Keyboard, and Micro SD Card* 5V and 3.3V regulators* Bankable EEPROM for 32k/32k RAM/ROM split or 64k ROM after boot* 99% Compatable with code designed for V1.X (UART Interrupt was removed)	Video coming sometime this summer...
